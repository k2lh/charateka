known issues

Currently tracked at github. If something breaks, errors, or doesn't work like you expect, check the Issues list and add what you've found. (You'll need a github account, but it's free and is easy to set up.) You can also see an up-to-date list of issues in the footer on every page, once you log in. Just scroll down to see.

what you can do

Upload characters in a batch, with ages and genders.

Track up to six stories.

Batch import events/scenes, created in a spreadsheet, or converted from a Scrivener or Aeon Timeline output.

Set character base information, and appearance and personality details.

Associate characters with stories, and set the character's role, motivation, goal, obstacle, and other notes.

Link characters as a direct hierarchial arrangement (ie employer), or list relationships in non-linked format (including to external characters not in the database).

Characters can also be grouped by free-form tags.

Associate characters with an event/scene as participants, observers, or as being spoken of.

Add notes for any changes to a character, as a result of an event.

Include notes for events, and view a compiled list as a story summary.

View a character's history for each story, showing participation in the events timeline.

View character demographics, overall and broken out by each story.
