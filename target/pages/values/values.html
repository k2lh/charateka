Events settings

Every event can also be tagged with an arc and a label. These can be whatever you want, and will appear in a drop-down when you create an event. Put one value on each row. (Do not include a comma at the end of a line, or it'll show up in the dropdown selections.)

Additional notes

By default, characters are displayed in the side-menu and all dropdowns with the 'full' name. If you don't enter a full name, the common (default) name will be used.

About protags values: this is a simple list, so if you change the order of your protagonists in this list, it will NOT change the values in any existing characters. Instead, you'll find that if you flipped the order of A and B, A now has the value for B, etc. If you want to edit protagonists, either add new ones at the end and ignore the rest, or be prepared to edit your existing characters.

Editing/adding values: make sure each value is on its own line. If a box is too small, some lines may wrap. As long as you don't have a 'return' in there, it counts as one line. You do NOT need to put a comma at the end of each line. This will just result in a value with a comma at the end. If you enter any html, it will not be output as html, but as the original code. The application isn't going to process that in drop-downs. Use utf-8 characters, instead. If you change a value after creating a character, this will NOT change the values for existing characters. Removing all values will not remove the option from the character creation; it will just give you an empty drop-down.

Please do not hose the database with malicious code. Thank you.
