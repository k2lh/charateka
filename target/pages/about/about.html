The Values page lets you enter and edit drop-down values. The only value options you can't change are gender and story role, because these are tied to specific icons. See all table icons on the Icons key page.

You can also import a list of characters names to get you started. You'll still need to go back and edit, but on the plus side, you'll be able to start linking characters right away. You can import the list on the Import page.

Notes about roles icons

Blue means the character is marked as an protagonist.

Red means the character is marked as an antagonist.

All other icons are just simple black, regardless of selected signifiers.

Any icon with a dot in the center means a POV character.

Antagonists are also identified with diamonds , instead of circles .

You can make background and off-page characters POV, but the icons won't change.
